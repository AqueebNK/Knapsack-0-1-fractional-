Pseudocode -

Initialize an empty list called items
Get references to the itemList, outputSection, and selectedItemsSection elements

When the "Add Item" button is clicked:
Get the name from the input field
Get the weight from the input field and convert it to an integer
Get the value from the input field and convert it to an integer

If name is not empty AND weight > 0 AND value > 0:
Create an object with name, weight, and value
Add the object to the items list
Call displayItems()

Clear the input fields for name, weight, and value

When the "Calculate" button is clicked:
Get the capacity from the input field and convert it to an integer
Call knapsack() with items and capacity
Call displayResults() with the selected items returned from knapsack()

Function displayItems:
Clear the itemList display
For each item in items with its index:
Create a new div displaying item's name, weight, and value

Create a delete button
Set its text to "Delete"
Add an event listener to call deleteItem() with the current index when clicked

Append the delete button to the div
Append the div to itemList

Function deleteItem(index):
Remove the item at the given index from items
Call displayItems() to update the display

Function knapsack(items, capacity):
Let n be the number of items
Create a 2D array dp with dimensions (n+1) x (capacity+1) initialized to 0

For i from 1 to n:
For w from 0 to capacity:
If the weight of the current item is less than or equal to w:
Update dp[i][w] with the maximum of:
dp[i-1][w] (not taking the item) OR dp[i-1][w - weight of current item] + value of current item (taking the item)
Else:
Set dp[i][w] to dp[i-1][w] (not taking the item)

Initialize w to capacity
Create an empty list called selectedItems

For i from n down to 1:
If dp[i][w] is not equal to dp[i-1][w]:
Add the current item to selectedItems
Decrease w by the weight of the current item

Return selectedItems

Function displayResults(selectedItems):
Clear the selectedItemsSection and outputSection displays
If selectedItems is empty: Set outputSection text to "No items selected."
Return

Initialize totalValue to 0

For each item in selectedItems:
Create a new div displaying item's name, weight, and value
Append the div to selectedItemsSection
Add item's value to totalValue

Set outputSection text to "Total Value: " followed by totalValue
