Pseudocode -

Initialize an empty list called items
When the "Add Item" button is clicked:
Get the item name from the input field
Get the item weight from the input field and convert it to a number
Get the item value from the input field and convert it to a number

If name is not empty AND weight > 0 AND value > 0:
Create an object with name, weight, value, and ratio (value / weight)
Add the object to the items list
Call displayItems()
Call clearInputs()
Else:
Show an alert: "Please enter valid item details."

When the "Calculate" button is clicked:
Get the capacity from the input field and convert it to a number
If capacity > 0:
Call fractionalKnapsack() with items and capacity
Display the maximum value and selected items in the output area
Else:
Show an alert: "Please enter a valid capacity."

Function displayItems:
Clear the item list display

For each item in items:
Create a new div displaying the item's name, weight, and value
Add a delete button with an index attribute
Append the div to the item list display

For each delete button:
When clicked:
Get the index from the button
Remove the item at that index from the items list
Call displayItems() to update the display

Function clearInputs:
Set the values of name, weight, and value input fields to empty strings

Function fractionalKnapsack(items, capacity):
Sort items by value-to-weight ratio in descending order

Initialize totalValue to 0
Initialize remainingCapacity to capacity
Initialize an empty list called selectedItems

For each item in items:
If remainingCapacity >= item.weight:
Add item.value to totalValue
Decrease remainingCapacity by item.weight
Add item.name to selectedItems
Else:
Calculate the fraction of the item that fits
Add value of the fraction to totalValue
Add item.name with the fraction to selectedItems
Break the loop

Return an object with maxValue and selectedItems

Time Complexity-

The dominant factor is the sorting step. Therefore, the overall time complexity of the fractional knapsack algorithm is: O(nlogn)
